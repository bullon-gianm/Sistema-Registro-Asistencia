# Sistema de Registro de Asistencia

Proyecto de desarrollo de software colaborativo utilizando Git, GitHub y metodologías ágiles.

## 👥 Integrantes del Equipo
* **Arturo(Líder/Estructura)** - Gestión de código, inicialización y ramas.
* **Arturo** - Ramas de componentes y desarrollo de módulos.
* **Ivis** - Automatización de procesos CI/CD.
* **Sandro** - Seguimiento y trazabilidad de Commits.

## 🔗 Enlace del Repositorio
* **GitHub Repository:** [https://github.com/bullon-gianm/Sistema-Registro-Asistencia.git]

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3, JavaScript.
* **Backend:** Node.js / Express.
* **Base de Datos:** SQL Server / MySQL.
* **DevOps:** GitHub Actions (CI/CD).
